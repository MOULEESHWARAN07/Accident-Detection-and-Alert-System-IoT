# 🚨 Accident Detection and Alert System

## 📌 Overview

Accident Detection and Alert System is an IoT-based vehicle safety project designed to automatically detect accidents and send emergency notifications.

The system uses an accelerometer sensor to monitor vehicle movement. When a sudden impact or abnormal motion is detected, the system identifies it as an accident and sends an alert message with GPS location details to emergency contacts using a GSM module.

This project helps reduce emergency response time and improves road safety.

---

## 🎯 Objectives

- Detect vehicle accidents automatically using sensors
- Monitor sudden impact using accelerometer data
- Get accident location using GPS module
- Send emergency alerts through GSM communication
- Reduce delay in accident reporting
- Improve safety and emergency response

## ⚙️ Working

1. Accelerometer continuously monitors vehicle movement.
2. Sudden impact or abnormal tilt is detected.
3. Arduino Nano processes sensor values.
4. Accident condition is verified using threshold values.
5. GPS module collects latitude and longitude.
6. GSM module sends SMS alert to emergency contacts.
